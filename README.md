# AI in Rheumatology Personal Brand Roadmap

This repository contains a simple personal website (`index.html`) that provides:

- A 12-month pathway to become a visible AI-in-rheumatology thought leader.
- A weekly operating plan.
- Portfolio project ideas.
- A built-in checklist with saved progress in your browser.

## Run locally

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.
